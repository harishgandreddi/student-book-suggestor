# 📚 Student Book Suggestor

A smart web application that recommends the right books for 
engineering students based on their branch, year, subject 
and difficulty level.


## 💡 Features
- Multi-step guided interface (Welcome → Year/Branch → Subject/Difficulty → Results)
- Book recommendations for CSE, ECE, Mechanical and Civil branches
- Filter by Year, Subject and Difficulty Level (Beginner/Intermediate/Advanced)
- Smart fallback — always shows books even if exact year has none
- Responsive design — works on mobile and desktop
- Clean dark theme UI

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## 📁 Project Structure
```
student-book-suggestor/
├── index.html     → Structure and layout
├── style.css      → Styling and dark theme
├── script.js      → Book data and filter logic
└── README.md      → Project documentation
```

## 🚀 How to Run
1. Clone the repository
2. Open `index.html` in any browser
3. No installation needed!
